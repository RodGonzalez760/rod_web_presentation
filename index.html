Portafolio

A continuación podrás revisar algunos proyectos pequeños en los que he trabajado de manera personal, con la finalidad de mostrar algunos conocimientos que he puesto en práctica. Todos los proyectos que se muestran a continuación fueron desarrollados con HTML 5, CSS 3 Y JAVASCRIPT, pero cada uno tiene una particularidad la que se describe en su descripción.

Mis Tweets localstorage

Este proyecto utiliza el localstorage del navegador para almacenar lo que escribimos en un textarea y lo muestra en una lista que se acumula, por lo que si se cierra la pestaña o el navegador, se mantendrán los 'tweets' que hemos escrito.

Busqueda por Múltiples Filtros

Este proyecto utiliza array methods como filter o foreach, encadenamiento de array methods, principios de programación funcional, funciones de alto nivel y eventos de javascript, manipulación del Dom y template de strings.

Buscador API Pixabay

Aquí utilizamos async/await y fetch API de Javascript para consumir la API externa de pixabay los resultados se muestran en tarjetas informativas con la opción de ver cada imagen en alta calidad, tiene un paginador creado con un Generador.

Cotizador de criptomonedas

Este proyecto utiliza async/await y fetch API de Javascript para consumir una API externa, muestra un spinner mientras se obtienen la información solicitada, y tiene la validación del formulario que muestra las alertas correspondientes.

Este proyecto es un administrador de citas, el cual tiene validación y limpieza del formulario después de cada creación de cita, permite editar y eliminar cada cita. Utiliza Clases de Javascript y el indexedDB del navegador.
